Detailed Description

template<typename TElement, int tDimension, typename TCudaType = CudaType<TElement>>
class bolt::TextureImage< TElement, tDimension, TCudaType >

Texture image representation, it owns data in a cudaArray and provides support for hardware interpolation using CUDA textureObject API
