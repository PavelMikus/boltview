Detailed Description

template<typename TImageType>
class bolt::mgt::ImageDescriptor< TImageType >

Helper class for storing image type and size.

This file contains implementations of memory storage objects for raw memory released from various image types. An image can be converted to its associated memory storage object and vice versa. A memory storage object is able to:

  • report the size of its owned memory in bytes
  • check if its owned memory is compatible with an image of requested type and size
