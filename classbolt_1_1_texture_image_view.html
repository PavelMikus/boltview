Detailed Description

template<typename TElement, int tDimension, typename TCudaType = DefaultCudaType<TElement>>
class bolt::TextureImageView< TElement, tDimension, TCudaType >

View to the whole texture image, which owns the data. It provides reference access to the data. It is usable on both host/device sides.
