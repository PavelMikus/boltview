Detailed Description

template<typename ... TBufferTypes>
class bolt::mgt::BufferAllocationWrapper< TBufferTypes >

Utility class wraps several buffer types defined as template parameters. It can allocate and estimate memory occupied in GpuMemoryManager, based on the parameters which would be passed to GpuMemoryManager. If the respectve buffer allocation would require more than one parameter (e.g. FFTCalculator) these must be bundled in a tuple.
