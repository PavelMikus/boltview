Detailed Description

template<typename TPixelTypeDst, typename TGilView>
class bolt::MaxRangeConverter< TPixelTypeDst, TGilView >

Converts between various instances of single-value channels (a.k.a "gray channels") by converting true source range into the data type destination range. This means that the converter first scans the source view to find the true min/max values, which are then converted to the full range of the output data type. This is unlike standard GIL converters, which convert between data type ranges. This is useful for example when dealing with float as a source, since GIL assumes float gray channels come from <0,1> range. This would require additional normalization on the computing site, which is not always desired.
