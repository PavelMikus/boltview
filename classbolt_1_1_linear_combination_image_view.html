Detailed Description

template<typename TFactor1, typename TView1, typename TFactor2, typename TView2>
class bolt::LinearCombinationImageView< TFactor1, TView1, TFactor2, TView2 >

Image view, which returns linear combination of elements from two other image views. R = f1 * I1 + f2 * I2
