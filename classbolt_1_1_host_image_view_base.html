Detailed Description

template<int tDimension, typename TPolicy = DefaultViewPolicy>
class bolt::HostImageViewBase< tDimension, TPolicy >

Base class for various image views. View concept should provide description (size) of the N-D image and provide access to its elements (pixels, voxels) through operator[]. Provides methods for size and coordinate queries.
