Detailed Description

template<typename TView, typename TInterpolator>
class bolt::InterpolatedView< TView, TInterpolator >

View which interpolates elements of another image view, using specified interpolator The interpolator must contain static function TView::Element compute(const TView &view, const Vector<float, TView::kDimension> &position)
