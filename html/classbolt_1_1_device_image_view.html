Detailed Description

template<typename TElement, int tDimension, typename TPolicy = DefaultViewPolicy>
class bolt::DeviceImageView< TElement, tDimension, TPolicy >

View to the part or whole device image, which owns the data. It provides reference access to the data. It is usable on both host/device sides.
