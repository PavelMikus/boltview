Detailed Description

template<typename TView, typename TInverseOperator>
class bolt::TransformedImageView< TView, TInverseOperator >

Provides transformed access to the provide image view

Template Parameters
TViewtype of transformed image view
TInverseOperatoroperator which provides access to original image from the transformed image, so it should implement inverse transformation. Called as inv(view, index, background_value), where view is the original view, index is element index in the transformed image view and background_value is used for elements leading outside from original image view.
