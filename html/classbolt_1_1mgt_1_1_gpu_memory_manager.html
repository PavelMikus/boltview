Detailed Description

GpuMemoryManager allocates gpu buffers on a single assigned gpu. Lifetime of the gpu buffers is managed by the GpuBufferView object. Instead of immediately freeing the images, their memory is stored in a cache and the memory manager then uses it to fulfill subsequent requests. TODO(tom) features:

  • buffer allocation via a unique id - allows sharing data between tasks
  • custom callbacks on deallocation - allows lazy copy out to main memory
