This file contains various implementations of a memory cache. A memory cache is able to:

  • store memory containers such as DeviceImagePointer from image_pointers.h
  • free the memory on request
  • return a compatible piece of memory for an image of desired type and size NOT threadsafe

Return maximal batch size, which would fit into the provided memory limit.

Parameters
firstsize of smallest possible batch
lastsize of largest possible batch
available_memorymemory in bytes
size_estimatorestimator function, which computes memory required by tested batch size
Returns
pair containing chosen batch size and amount of memory it would need. If no such batch exists, it returns {-1, -1}.
