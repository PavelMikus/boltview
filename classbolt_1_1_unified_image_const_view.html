Detailed Description

template<typename TElement, int tDimension, typename TPolicy = DefaultViewPolicy>
class bolt::UnifiedImageConstView< TElement, tDimension, TPolicy >

View to the part or whole unified-memory image, which owns the data. It provides only constant access to the data. It is usable on both host/device sides.
